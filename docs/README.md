## 🛤️ Треки обучения

### 🎯 Трек 1: Junior Full-Stack Developer
1. **Fundamentals** → **Frontend** (Foundations) → **Backend** → **Technical Skills** (Testing)

### 🎯 Трек 2: Senior Frontend Developer
1. **Fundamentals** → **Frontend** (полностью) → **Algorithms** → **Architecture**

### 🎯 Трек 3: Backend/API Developer
1. **Fundamentals** → **Backend** → **Technical Skills** → **Architecture** → **Infrastructure**

### 🎯 Трек 4: DevOps Engineer
1. **Technical Skills** (Systems) → **Infrastructure** → **Backend** → **Architecture**

### 🎯 Трек 5: ML/AI Engineer
1. **Mathematics** → **AI & LLM** → **Algorithms** → **Backend** → **Infrastructure**

### 🎯 Трек 6: System Architect
1. **Computer Science** → **Architecture** → **Technical Skills** → **Infrastructure** → **Leadership**

### 🎯 Трек 7: Technical Lead
1. **Algorithms** → **Architecture** → **Leadership** → **Technical Skills** → **Infrastructure**

### 🎯 Трек 8: Data Scientist
1. **Mathematics** → **AI & LLM** → **Algorithms** → **Languages** (Python/R)
