# Самоучитель: Хирагана и Катакана

## 🎯 Цель курса
Полное освоение японских силлабических азбук за 2-4 недели с возможностью читать и писать любые японские слова в хирагане и катакане.

## 📚 Структура обучения

### Неделя 1: Основы хираганы (あ-row to な-row)
```
День 1-2: あ-ряд (a-ряд)
あ (a) い (i) う (u) え (e) お (o)

Мнемоники:
あ - похоже на "а" с закругленными линиями
い - как буква "и" с двумя штрихами  
う - похоже на "у" с крючком
え - как "е" с дополнительной линией
お - как "о" с пересекающимися линиями

Упражнения День 1:
1. Обведите каждый символ 20 раз
2. Напишите без образца 10 раз каждый
3. Прочитайте вслух: あいうえお
4. Запишите под диктовку: あ, い, う, え, お

Слова для практики:
愛 (あい) - любовь
家 (いえ) - дом
上 (うえ) - верх
絵 (え) - картина
青 (あお) - синий
```

```
День 3-4: か-ряд (k-ряд)
か (ka) き (ki) く (ku) け (ke) こ (ko)

Произношение:
か - [ka] как в слове "какао"
き - [ki] как в слове "киви"  
く - [ku] как в слове "кукла"
け - [ke] как в слове "кеды"
こ - [ko] как в слове "кот"

Упражнения День 3:
1. Напишите か-ряд 15 раз
2. Комбинируйте с あ-рядом: かい, きい, くう
3. Читайте слова: かき (хурма), ここ (здесь)
4. Диктант: смешанные символы あ-か рядов

Новые слова:
書く (かく) - писать
聞く (きく) - слушать
九 (きゅう/く) - девять
毛 (け) - волосы
声 (こえ) - голос
```

```
День 5-7: さ-ряд (s-ряд)
さ (sa) し (shi) す (su) せ (se) そ (so)

Особенности произношения:
し - [shi] НЕ [si]! Важное отличие
す - [su] мягкое "с", почти без гласной в конце слов

Упражнения День 5:
1. Отдельная практика し - очень важный звук
2. Слова с し: すし (суши), しお (соль)
3. Различение し и き в словах
4. Написание коротких фраз: これです (это есть)

Практические слова:
寿司 (すし) - суши
塩 (しお) - соль
背 (せ) - спина
空 (そら) - небо
```

```
День 6-7: た-ряд (t-ряд) и な-ряд (n-ряд)
た (ta) ち (chi) つ (tsu) て (te) と (to)
な (na) に (ni) ぬ (nu) ね (ne) の (no)

Сложные звуки:
ち - [chi] НЕ [ti]
つ - [tsu] уникальный японский звук

Упражнения:
1. Специальная практика ち и つ
2. Комбинации: ちち (папа), つつ (обертывать)
3. Частица の: わたしの (мой/моя)
4. Простые предложения: これはなにですか

Ключевые слова:
父 (ちち) - отец
手 (て) - рука
人 (ひと) - человек
犬 (いぬ) - собака
猫 (ねこ) - кошка
```

### Неделя 2: Завершение хираганы
```
День 8-9: は-ряд, ま-ряд, や-ряд
は (ha) ひ (hi) ふ (fu) へ (he) ほ (ho)
ま (ma) み (mi) む (mu) め (me) も (mo)
や (ya) ゆ (yu) よ (yo)

Важные частицы:
は - читается [wa] как частица темы
へ - читается [e] как частица направления
を - читается [o] как частица объекта

Упражнения:
1. Различение は как [ha] и как [wa]
2. Простые предложения с частицами
3. Слова: はは (мама), ふね (корабль)

Грамматические примеры:
わたしはがくせいです。(Я студент)
がっこうへいきます。(Иду в школу)
ほんをよみます。(Читаю книгу)
```

```
День 10-11: ら-ряд, わ-ряд, ん
ら (ra) り (ri) る (ru) れ (re) ろ (ro)
わ (wa) ゐ (wi) ゑ (we) を (wo/o) ん (n)

Произношение ら-ряда:
Японское "r" - среднее между "r" и "l"
Кончик языка быстро касается неба

Упражнения:
1. Специальная практика ら-звуков
2. Слова: らーめん (рамен), りんご (яблоко)
3. ん в разных позициях: ほん (книга), せんせい (учитель)

Полезные слова:
林檎 (りんご) - яблоко
本 (ほん) - книга
先生 (せんせい) - учитель
```

```
День 12-14: Комбинации (dakuten, handakuten, yōon)
が ぎ ぐ げ ご (g-звуки)
ざ じ ず ぜ ぞ (z-звуки)  
だ ぢ づ で ど (d-звуки)
ば び ぶ べ ぼ (b-звуки)
ぱ ぴ ぷ ぺ ぽ (p-звуки)

きゃ きゅ きょ (kya, kyu, kyo)
しゃ しゅ しょ (sha, shu, sho)
ちゃ ちゅ ちょ (cha, chu, cho)
にゃ にゅ にょ (nya, nyu, nyo)
ひゃ ひゅ ひょ (hya, hyu, hyo)
みゃ みゅ みょ (mya, myu, myo)
りゃ りゅ りょ (rya, ryu, ryo)
ぎゃ ぎゅ ぎょ (gya, gyu, gyo)
じゃ じゅ じょ (ja, ju, jo)
びゃ びゅ びょ (bya, byu, byo)
ぴゃ ぴゅ ぴょ (pya, pyu, pyo)

Упражнения:
1. Каждая комбинация 10 раз письменно
2. Чтение слов: ちゃ (чай), しゅくだい (домашнее задание)
3. Различение じ/ぢ и ず/づ

Практические слова:
茶 (ちゃ) - чай  
宿題 (しゅくだい) - домашнее задание
料理 (りょうり) - кулинария
病院 (びょういん) - больница
```

### Неделя 3: Катакана основы
```
День 15-17: Основная катакана (ア-ナ)
ア (a) イ (i) ウ (u) エ (e) オ (o)
カ (ka) キ (ki) ク (ku) ケ (ke) コ (ko)
サ (sa) シ (shi) ス (su) セ (se) ソ (so)
タ (ta) チ (chi) ツ (tsu) テ (te)ト (to)
ナ (na) ニ (ni) ヌ (nu) ネ (ne) ノ (no)

Отличия от хираганы:
- Более угловатые формы
- Используется для заимствованных слов
- Иностранные имена
- Звукоподражания

Примеры слов:
コーヒー (kōhī) - кофе
テレビ (terebi) - телевизор  
コンピューター (konpyūtā) - компьютер
アメリカ (Amerika) - Америка
レストラン (resutoran) - ресторан
```

```
День 18-19: Завершение катаканы (ハ-ワ)
ハ (ha) ヒ (hi) フ (fu) ヘ (he) ホ (ho)
マ (ma) ミ (mi) ム (mu) メ (me) モ (mo)
ヤ (ya) ユ (yu) ヨ (yo)
ラ (ra) リ (ri) ル (ru) レ (re) ロ (ro)
ワ (wa) ヲ (wo) ン (n)

Длинные гласные в катакане:
ー используется для продления звука
コーヒー (длинное o)
ケーキ (длинное e)
スーパー (длинное u и a)

Упражнения:
1. Написание имен: ロシア (Россия), マリア (Мария)
2. Еда: パン (хлеб), バナナ (банан), パスタ (паста)
3. Технологии: インターネット (интернет), メール (email)

Полезная лексика:
パン - хлеб
バナナ - банан  
パスタ - паста
インターネット - интернет
メール - email
ホテル - отель
```

```
День 20-21: Специальные комбинации катаканы
ガ ギ グ ゲ ゴ (g-звуки)
ザ ジ ズ ゼ ゾ (z-звуки)
ダ ヂ ヅ デ ド (d-звуки)  
バ ビ ブ ベ ボ (b-звуки)
パ ピ プ ペ ポ (p-звуки)

Специальные комбинации для иностранных звуков:
ファ (fa) フィ (fi) フェ (fe) フォ (fo)
ティ (ti) ディ (di)
トゥ (tu) ドゥ (du)
ウィ (wi) ウェ (we) ウォ (wo)
ヴァ (va) ヴィ (vi) ヴェ (ve) ヴォ (vo)

Упражнения:
1. Иностранные имена: デイヴィッド (David), ティファニー (Tiffany)
2. Бренды: マクドナルド (McDonald's), ディズニー (Disney)
3. Спорт: フットボール (футбол), バスケットボール (баскетбол)

Современная лексика:
デイヴィッド - David
ティファニー - Tiffany
マクドナルド - McDonald's
ディズニー - Disney
フットボール - футбол
バスケットボール - баскетбол
```

## 🎯 Практические советы

### Мнемонические техники
```
Хирагана-истории:
あ - "А, какая красивая арка!"
き - "Ключ с двумя зубчиками"
す - "Спираль закручивается"
ね - "Нерв извивается как змея"
ほ - "Человек поднял руку и ногу"

Катакана-ассоциации:
ア - "Индейский вигвам" (треугольная форма)
カ - "Нож режет" (острые углы)
ス - "Скользящая лыжа"
ト - "Тотемный столб"
フ - "Флаг развевается"
```

### Частые ошибки и как их избежать
```
Хирагана:
❌ Путаница は (ha) и ほ (ho)
✅ は - круг справа, ほ - линия проходит через круг

❌ Неправильный порядок штрихов
✅ Всегда слева направо, сверху вниз

❌ Забывают о маленьких つ っ
✅ っ = удвоение следующего согласного

Катакана:
❌ Путаница シ (shi) и ツ (tsu)  
✅ シ - штрихи идут вниз, ツ - горизонтально

❌ Неправильное использование ー
✅ Только для продления звука в катакане

❌ Забывают специальные комбинации
✅ ファ, ティ, ディ для иностранных слов
```

### Каллиграфия и красивое письмо
```
Принципы красивой каны:
1. Правильный порядок штрихов
2. Равномерный размер символов
3. Одинаковые интервалы
4. Плавные, уверенные линии
5. Правильный наклон (немного вправо)

Инструменты:
- Специальные прописи (練習帳)
- Карандаш или тонкая ручка
- Клетчатая бумага (genkouyoushi)
- Образцы для копирования

Ежедневная каллиграфия (10 минут):
- 5 символов особенно красиво
- Полная концентрация на форме
- Медленно и осознанно
- Сравнение с образцом
```
