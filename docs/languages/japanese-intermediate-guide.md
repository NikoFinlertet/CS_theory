# Самоучитель: Средний уровень японского языка (Phase 2)

## 🎯 Цель этапа: Intermediate Development (1500-4000 часов)

**Временные рамки:** 18-24 месяца при 4-5 часах ежедневной практики  
**Предварительные требования:** Завершенный Phase 1 (хирагана, катакана, 500+ кандзи, базовая грамматика)  
**Результат:** Средний уровень владения японским языком (N3-N2 JLPT)

## 📚 План 1: Продвинутая грамматика

### Месяц 1-3: Кэйго (Keigo) - Вежливый язык

#### Теизай-го (Teinei-go) - Стандартная вежливость
```
Уже изученные формы:
です/である (desu/de aru) - связка
ます (masu) - вежливое окончание глаголов

Расширение:
でございます (de gozaimasu) - очень вежливая связка
いらっしゃいます (irasshaimasu) - вежливая форма "есть/находиться"

Примеры использования:
お名前は何でございますか。
(O-namae wa nan de gozaimasu ka?)
- Как ваше имя? (очень вежливо)

先生はお元気でいらっしゃいますか。
(Sensei wa o-genki de irasshaimasu ka?)
- Учитель здоров? (вежливо о третьем лице)

Упражнения:
1. Переведите обычные предложения в вежливую форму
2. Создайте диалог в магазине с использованием keigo
3. Практикуйте формальные приветствия
```

#### Сонкэй-го (Sonkei-go) - Почтительный язык
```
Основные формы уважения к собеседнику:

お/ご + существительное:
お名前 (o-namae) - ваше имя
ご家族 (go-kazoku) - ваша семья
お仕事 (o-shigoto) - ваша работа

Почтительные глаголы:
いらっしゃる (irassharu) - быть, идти, приходить
なさる (nasaru) - делать
おっしゃる (ossharu) - говорить, сказать
ご覧になる (goran ni naru) - смотреть
召し上がる (meshiagaru) - есть, пить

Примеры в контексте:
部長はもういらっしゃいましたか。
- Руководитель отдела уже пришел?

何をなさっていますか。
- Что вы делаете?

先生がおっしゃったことを覚えています。
- Я помню то, что сказал учитель.

Ежедневная практика:
- 15 минут почтительных форм
- Диалоги с начальником/учителем
- Формальные ситуации
```

#### Кэнджё-го (Kenjou-go) - Скромный язык
```
Принижение собственных действий:

Основные скромные глаголы:
参る (mairu) - идти, приходить
いたす (itasu) - делать
申す (mousu) - говорить, называться
拝見する (haiken suru) - смотреть
いただく (itadaku) - получать, есть, пить

Конструкции со скромностью:
お/ご + основа глагола + する:
お手伝いする (o-tetsudai suru) - помогать
ご説明する (go-setsumei suru) - объяснять
お待ちする (o-machi suru) - ждать

Практические примеры:
明日お伺いいたします。
- Завтра приду к вам (скромно).

資料を拝見させていただきました。
- Посмотрел ваши материалы (очень скромно).

お名前をお聞かせください。
- Скажите, пожалуйста, ваше имя (скромная просьба).

Культурный контекст:
- Самопринижение как форма уважения
- Различие между внутренней и внешней группой
- Ситуации использования
```

### Месяц 4-6: Условные конструкции

#### Различные типы "если"
```
1. と (to) - естественное следствие:
雨が降ると道が濡れる。
- Если идет дождь, дорога мокреет.

春になると桜が咲く。
- Когда приходит весна, цветет сакура.

2. ば (ba) - условие-предположение:
時間があれば映画を見ます。
- Если будет время, посмотрю фильм.

もっと勉強すれば合格できます。
- Если будешь больше учиться, сможешь сдать экзамен.

3. たら (tara) - конкретное условие:
家に帰ったら電話します。
- Когда приду домой, позвоню.

お金があったら旅行したい。
- Если бы были деньги, хотел бы путешествовать.

4. なら (nara) - если говорить о...:
日本語なら分かります。
- Если о японском языке, то понимаю.

行くなら一緒に行きましょう。
- Если идете, пойдемте вместе.

Упражнения на различия:
1. Переведите с правильной условной формой
2. Объясните разницу между と и たら
3. Создайте диалоги с каждой формой
```

#### Сложные условные конструкции
```
Условие + цель:
図書館に行けば本が借りられます。
- Если пойдешь в библиотеку, сможешь взять книги.

Противоположные условия:
雨が降っても行きます。
- Даже если пойдет дождь, пойду.

雨が降らなくても傘を持って行きます。
- Даже если дождя не будет, возьму зонт.

Множественные условия:
時間があって、お金があれば、旅行します。
- Если будет время И деньги, поеду в путешествие.

Нереальные условия:
もし宝くじに当たったら何をしますか。
- Что бы вы делали, если бы выиграли в лотерею?

Практические ситуации:
- Планирование поездок
- Обсуждение карьеры
- Житейские ситуации
```

### Месяц 7-9: Причинно-следственные связи

#### Выражение причины
```
1. から (kara) - потому что (субъективная причина):
疲れているから早く寝ます。
- Поскольку устал, рано лягу спать.

日本語が難しいから時間がかかります。
- Поскольку японский трудный, требуется время.

2. ので (no de) - поэтому (объективная причина):
雨が降っているので傘を持って行きます。
- Поскольку идет дождь, возьму зонт.

電車が遅れているので遅刻します。
- Поскольку поезд опаздывает, опоздаю.

3. ため(に) (tame ni) - для того чтобы, из-за:
健康のために運動しています。
- Занимаюсь спортом для здоровья.

事故のため電車が止まりました。
- Из-за аварии поезд остановился.

Нюансы использования:
- から - личное мнение, эмоции
- ので - факты, вежливость
- ため - цель или серьезная причина
```

#### Выражение результата
```
Конструкции результата:

その結果 (sono kekka) - в результате:
勉強した。その結果、合格した。
- Учился. В результате сдал экзамен.

そのため (sono tame) - поэтому:
雨が降った。そのため試合が中止になった。
- Пошел дождь. Поэтому матч отменили.

だから (dakara) - поэтому (разговорное):
忙しい。だから行けない。
- Занят. Поэтому не могу пойти.

それで (sore de) - и тогда, поэтому:
電車が遅れた。それで遅刻した。
- Поезд опоздал. И поэтому опоздал.

Практика логического мышления:
1. Анализ причинно-следственных связей
2. Построение аргументации
3. Объяснение ситуаций
```

### Месяц 10-12: Модальность и намерения

#### Выражение намерений
```
1. つもり (tsumori) - намерение:
来年日本に行くつもりです。
- Намереваюсь поехать в Японию в следующем году.

もう帰るつもりはありません。
- Не намереваюсь больше возвращаться.

2. 予定 (yotei) - план:
来週会議の予定です。
- На следующей неделе планируется совещание.

3. はず (hazu) - должно быть, ожидается:
彼はもう来ているはずです。
- Он уже должен быть здесь.

この薬を飲めば治るはずです。
- Если примешь это лекарство, должно выздороветь.

4. だろう/でしょう - вероятно:
明日は雨でしょう。
- Завтра, вероятно, будет дождь.

彼は来ないだろう。
- Он, вероятно, не придет.

Градации уверенности:
確実 (kakujitsu) - уверенность
たぶん (tabun) - вероятно
かもしれない (kamoshirenai) - может быть
```

#### Выражение возможности и способности
```
1. ことができる (koto ga dekiru) - мочь:
日本語を話すことができます。
- Могу говорить по-японски.

2. Потенциальная форма глаголов:
Group 1: 読む → 読める (yomu → yomeru)
Group 2: 食べる → 食べられる (taberu → taberareru)
Group 3: する → できる, 来る → 来られる

3. やすい/にくい - легко/трудно:
この本は読みやすいです。
- Эту книгу легко читать.

漢字は覚えにくいです。
- Кандзи трудно запоминать.

4. かもしれない - возможность:
雨が降るかもしれません。
- Возможно, пойдет дождь.

彼は病気かもしれません。
- Возможно, он болен.

Практические упражнения:
1. Описание своих возможностей
2. Выражение планов на будущее
3. Обсуждение вероятностей
```

## 📖 План 2: Расширение кандзи (500 → 1500)

### Систематическое изучение радикалов

#### Основные радикалы и их значения
```
Человек (人系):
人 (человек) → 仕事 (работа), 体 (тело)
亻(にんべん) → 何 (что), 住 (жить)
彳(ぎょうにんべん) → 行 (идти), 街 (улица)

Вода (水系):
水 (вода) → 海 (море), 泳 (плавать)
氵(さんずい) → 河 (река), 洗 (мыть)
冫(にすい) → 冷 (холодный), 凍 (замерзать)

Дерево (木系):
木 (дерево) → 森 (лес), 机 (стол)
朩(きへん) → 校 (школа), 検 (проверять)

Огонь (火系):
火 (огонь) → 燃 (гореть), 炊 (готовить)
灬(れっか) → 熱 (горячий), 煮 (варить)

Сердце (心系):
心 (сердце) → 思 (думать), 意 (смысл)
忄(りっしんべん) → 快 (приятный), 情 (чувство)

Методика изучения:
1. Группируйте кандзи по радикалам
2. Ищите логические связи в значениях
3. Используйте мнемонические ассоциации
4. Практикуйте комбинации радикалов
```

## 📚 План 3: Навыки чтения

### Месяц 1-6: Развитие беглости чтения

#### Стратегии понимания текста
```
1. Скимминг (поверхностное чтение):
Цель: Понять общую идею текста
Время: 1-2 минуты на страницу
Техника:
- Читайте только первые предложения абзацев
- Ищите ключевые слова и имена
- Не останавливайтесь на незнакомых кандзи
- Формируйте общее представление

Практика скимминга:
新聞記事 (статьи в газетах)
雑誌 (журнальные статьи)
ブログ記事 (блог-посты)
ウェブサイト (веб-сайты)

2. Сканирование (поиск информации):
Цель: Найти конкретную информацию
Техника:
- Определите, что ищете
- Пропускайте нерелевантные части
- Фокусируйтесь на ключевых словах
- Используйте структуру текста

Примеры задач:
- Найти время встречи в письме
- Найти цену в объявлении
- Найти адрес в инструкции
- Найти дату в новости

3. Интенсивное чтение:
Цель: Полное понимание
Техника:
- Читайте медленно и внимательно
- Анализируйте незнакомые слова
- Переводите сложные предложения
- Делайте заметки
```

#### Работа с различными типами текстов

##### Художественная литература
```
Начальный уровень:
- 子供向けの本 (детские книги)
- 短編小説 (короткие рассказы)
- 漫画 (манга с фуриганой)

Средний уровень:
- 青春小説 (молодежные романы)
- ミステリー (детективы)
- 恋愛小説 (романтические истории)

Техники чтения художественной литературы:
1. Контекстное понимание:
知らない単語があっても、文脈から意味を推測する
- Даже если есть незнакомые слова, угадывайте значение по контексту

2. Эмоциональное понимание:
キャラクターの気持ちを理解する
- Понимайте чувства персонажей

3. Культурные особенности:
日本の文化的背景を学ぶ
- Изучайте японский культурный контекст

Еженедельные цели:
- 20-30 страниц художественного текста
- 1 короткий рассказ
- 2-3 главы манги
- Ведение читательского дневника
```

##### Нехудожественная литература
```
Информационные тексты:
- 新聞 (газеты) - политика, экономика, спорт
- 雑誌 (журналы) - наука, технологии, хобби
- 教科書 (учебники) - академические тексты
- 説明書 (инструкции) - технические тексты

Стратегии для нехудожественных текстов:

1. Предварительное изучение:
- Просмотрите заголовки и подзаголовки
- Изучите диаграммы и картинки
- Прочитайте введение и заключение
- Определите структуру текста

2. Активное чтение:
- Задавайте вопросы тексту
- Делайте заметки на полях
- Выделяйте ключевые факты
- Связывайте с предыдущими знаниями

3. После чтения:
- Перескажите основные идеи
- Создайте схему или карту понятий
- Обсудите с другими
- Примените новые знания

Специализированная лексика:
医学用語 (медицинские термины)
法律用語 (юридические термины)  
科学用語 (научные термины)
経済用語 (экономические термины)
```

### Месяц 7-12: Критическое чтение

#### Анализ аргументации
```
Элементы аргументации в японских текстах:

1. 主張 (shuchou) - утверждение:
筆者の主な意見を特定する
- Определите основное мнение автора

2. 根拠 (konkyo) - основания:
主張を支える証拠や理由を探す
- Ищите доказательства и причины, поддерживающие утверждение

3. 例 (rei) - примеры:
具体的な事例で理解を深める
- Углубляйте понимание конкретными примерами

4. 反証 (hanshō) - контраргументы:
反対意見や問題点を考慮する
- Рассматривайте противоположные мнения и проблемы

Типичные структуры аргументации:

起承転結 (ki-shō-ten-ketsu):
起 - введение темы
承 - развитие темы
転 - поворот или контраст
結 - заключение

Практические упражнения:
```

## 🗣️ План 4: Культурная компетентность

### Понимание японского менталитета

#### Концепция группы vs индивидуальности
```
内 (uchi) vs 外 (soto) - внутренняя vs внешняя группа:

Внутренняя группа (身内 - miuchi):
- Семья
- Коллеги по работе
- Одноклассники
- Близкие друзья

Внешняя группа (他人 - tanin):
- Незнакомцы
- Другие компании
- Иностранцы

Языковые различия:
内での話し方 (uchi de no hanashikata):
- Более прямая речь
- Меньше кэйго
- Больше эмоций

外での話し方 (soto de no hanashikata):
- Формальная речь
- Полное кэйго
- Сдержанность

Практические ситуации:
1. В офисе с коллегами vs с клиентами
2. Дома vs в публичных местах
3. С японцами vs с иностранцами
4. В школе vs вне школы

Упражнения:
- Ролевые игры в разных ситуациях
- Анализ поведения в драмах
- Обсуждение границ групп
- Практика переключения стилей
```

```
Жесты и позы:

手の位置 (te no ichi) - положение рук:
- При разговоре
- При слушании
- При ожидании

Выражения лица:
- Сдержанность эмоций

Личное пространство:

```
