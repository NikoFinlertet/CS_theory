# Английский язык: Диалектология и языковая вариативность

## 🌍 Основы диалектологии

### Понятие языковой вариативности
```
Типы вариации:
├── Географическая (диатопическая): regional variation
├── Социальная (диастратическая): social variation
├── Ситуативная (диафазическая): stylistic variation
├── Временная (диахроническая): historical variation
├── Медийная (диамедиальная): medium variation
├── Идиолектная: individual variation
└── Групповая: group variation

Лингвистические уровни:
├── Фонетический: произношение
├── Фонологический: звуковые системы
├── Лексический: словарный состав
├── Морфологический: формы слов
├── Синтаксический: структура предложений
├── Семантический: значения
└── Прагматический: использование
```

### Фонетические особенности
```
Сдвиг северных городов (Northern Cities Shift):
├── /æ/ → [eə] в словах типа "cat"
├── /ɑ/ → [a] в словах типа "caught"
├── /ɔ/ → [ɑ] в словах типа "cot"
├── /ɛ/ → [ɪ] в словах типа "bet"
├── /ɪ/ → [e] в словах типа "bit"
└── /ʌ/ → [ɔ] в словах типа "but"

Южные особенности:
├── Монофтонгизация /aɪ/ → [a:]
├── Дифтонгизация /ɛ/ → [eə]
├── Пин-пен слияние: /ɪ/ = /ɛ/ перед /n/
├── R-dropping в старых диалектах
├── Различные реализации /aʊ/
└── Интонационные особенности

Западные особенности:
├── Кот-каут слияние: /ɑ/ = /ɔ/
├── Калифорнийский гласный сдвиг
├── Особенности /æ/
├── Реализации /ɛ/
└── Интонационные паттерны
```
