# Английский язык: Психолингвистика и когнитивные аспекты

## 🧠 Основы психолингвистики

### Предмет и задачи
```
Определение психолингвистики:
├── Междисциплинарная область знания
├── Изучение психических процессов в языке
├── Связь между языком и мышлением
├── Когнитивные механизмы обработки языка
└── Нейробиологические основы языка

Основные вопросы:
├── Как мозг обрабатывает язык?
├── Как происходит понимание речи?
├── Как формируется языковая способность?
├── Как изучается второй язык?
└── Что происходит при языковых расстройствах?
```

### Методы исследования
```
Поведенческие методы:
├── Время реакции (RT): быстрота ответа
├── Процент ошибок (accuracy): точность
├── Прайминг: влияние контекста
├── Лексическое решение: слово/не-слово
├── Кросс-модальный прайминг
├── Gating paradigm: постепенное предъявление
├── Eye-tracking: движения глаз
└── Самопериодическое чтение: reading time

Нейрофизиологические методы:
├── ERP: вызванные потенциалы
├── fMRI: функциональная томография
├── PET: позитронно-эмиссионная томография
├── MEG: магнитоэнцефалография
├── TMS: транскраниальная стимуляция
├── Single-cell recording: отдельные нейроны
├── Optogenetics: оптическая стимуляция
└── Интракорковые записи: электроды в мозге
```

### Теоретические модели
```
Основные подходы:
├── Модульный подход: независимые системы
├── Интерактивный подход: взаимодействие уровней
├── Коннекционистский подход: нейронные сети
├── Символический подход: правила и представления
├── Статистический подход: вероятностные модели
├── Воплощенный подход: телесная основа
└── Динамический подход: самоорганизация

Классические модели:
├── Модель Фодора: модульность
├── Модель Марслена-Вильсона: COHORT
├── Модель Маккелланда-Элмана: TRACE
├── Модель Делла: продукция речи
├── Модель Левельта: говорение
└── Модель Колтхарта: чтение
```

## 👂 Восприятие и понимание речи

### Восприятие звуков речи
```
Категориальное восприятие:
├── Границы между фонемами
├── Дискриминация внутри/между категориями
├── Кривая идентификации
├── Эффект Мак-Гурка: аудиовизуальная интеграция
├── Фонематическая реставрация
└── Адаптация к акценту

Нейронные механизмы:
├── Первичная слуховая кора (A1)
├── Верхняя височная извилина (STG)
├── Верхняя височная борозда (STS)
├── Область Вернике: понимание речи
├── Моторная кора: артикуляторные репрезентации
├── Зеркальные нейроны: восприятие-действие
└── Предсказательное кодирование
```

### Распознавание слов
```
Процесс распознавания:
├── Фонетическая обработка: звуки → фонемы
├── Фонологическая обработка: фонемы → слова
├── Лексический доступ: активация кандидатов
├── Отбор: выбор правильного слова
├── Интеграция: вставка в контекст
└── Мониторинг: проверка правильности

Модели распознавания:
├── COHORT model:
│   ├── Активация когорты кандидатов
│   ├── Отбор по фонетическим признакам
│   ├── Интеграция семантической информации
│   └── Единственный победитель
├── TRACE model:
│   ├── Интерактивная активация
│   ├── Три уровня: признаки, фонемы, слова
│   ├── Возбуждение и торможение
│   └── Параллельная обработка
└── Shortlist model:
    ├── Фонологическая когорта
    ├── Соревнование кандидатов
    ├── Частотные эффекты
    └── Отложенная активация
```

## 🧒 Усвоение языка

### Первичное усвоение
```
Этапы развития:
├── Преддощельный период (0-6 мес):
│   ├── Категориальное восприятие
│   ├── Просодическая чувствительность
│   ├── Распознавание голоса матери
│   ├── Предпочтение речи
│   └── Развитие фонематических границ
├── Дощельный период (6-12 мес):
│   ├── Каноническое дщение
│   ├── Лексическое сегментирование
│   ├── Фонологическое развитие
│   ├── Жестовая коммуникация
│   └── Понимание слов
├── Однословный период (12-18 мес):
│   ├── Первые слова
│   ├── Голофразы
│   ├── Понимание vs продукция
│   ├── Избыточное расширение
│   └── Недостаточное расширение
├── Двусловный период (18-24 мес):
│   ├── Телеграфная речь
│   ├── Семантические роли
│   ├── Словарный взрыв
│   ├── Грамматикализация

└── Многословный период (2-5 лет):

```

├── Единая система: shared system

├── Неселективный доступ: non-selective access
├── Конвергентные области: convergent areas
├── Дивергентные области: divergent areas
├── Возрастные эффекты: age effects
└── Уровень владения: proficiency effects

Процессы обработки:
├── Переключение языков: code-switching
├── Интерференция: cross-linguistic influence
├── Ингибирование: inhibition
├── Активация: activation
├── Контроль: cognitive control
├── Мониторинг: monitoring
└── Конфликт: conflict resolution
```

### Модели билингвизма
```
Ранние модели:
├── Модель координации: coordinate model
├── Модель соединения: compound model
├── Модель субординации: subordinate model
├── Пороговая гипотеза: threshold hypothesis
├── Развивающаяся взаимозависимость

Современные модели:
├── BIA model: bilingual interactive activation

├── RHM model: revised hierarchical model
├── SOMBIP model: self-organizing model
