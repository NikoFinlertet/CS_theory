# Деловой японский язык: Полный практический курс

## 🎯 Цель курса: Профессиональное владение деловым японским

**Предварительные требования:** Уровень N3-N2 JLPT, базовое понимание кэйго  
**Продолжительность:** 12-18 месяцев интенсивного изучения

#### ビジネスマナー (Деловой этикет)
```
名刺交換 (meishi koukan) - обмен визитками:

基本手順:
1. 両手で差し出す - подавать двумя руками
2. 「よろしくお願いいたします」と言う - говорить приветствие
3. 相手の名刺を丁寧に受け取る - аккуратно принимать
4. しばらく眺めて内容を確認 - внимательно изучать
5. テーブルの上に整列して置く - аккуратно класть на стол

重要な表現:
初めてお目にかかります。
- Рад познакомиться (первая встреча).

いつもお世話になっております。
- Спасибо за постоянную заботу (повторная встреча).

会議のマナー (Этикет совещаний):

座席の順序 (рассадка):
- 入口から最も遠い席 = 上座 (kamiza) - почетное место
- 入口に最も近い席 = 下座 (shimoza) - скромное место
- 年長者・上司が上座に座る

発言の順序:
1. 司会者の進行に従う - следовать ведущему
2. 上司から意見を求められてから発言 - говорить после приглашения
3. 反対意見は慎重に表現 - осторожно выражать несогласие
4. 「恐れ入りますが」で始める - начинать с извинения

Практические фразы:
貴重なご意見をありがとうございます。
- Спасибо за ценное мнение.

ちょっと質問があるのですが...
- У меня есть небольшой вопрос...

お時間をいただいてすみません。
- Извините за потраченное время.
```

### コミュニケーションスタイル

#### 間接的表現 (Косвенные выражения)
```
断り方 (способы отказа):

直接的 (прямо): だめです - плохо/нельзя
間接的 (косвенно): ちょっと難しいですね... - это немного сложно...

ビジネスでの断り方:
検討させていただきます。
- Позвольте рассмотреть это. (вежливый отказ)

社内で相談してから回答いたします。
- Ответим после внутреннего обсуждения.

スケジュールを確認してご連絡します。
- Свяжемся после проверки расписания.

提案の仕方 (способы предложений):

いかがでしょうか - как насчет этого?
～してはどうでしょうか - как насчет того, чтобы...?
もしよろしければ - если не возражаете...

例:
来週お時間をいただけるでしょうか。
- Могли бы вы уделить время на следующей неделе?

この案件について検討していただけませんか。
- Не могли бы вы рассмотреть этот проект?

意見の表明 (выражение мнений):

個人的には... - лично я думаю...
私の考えでは... - по моему мнению...
率直に申し上げますと... - откровенно говоря...

注意深い表現:
もしかすると... - возможно...
おそらく... - вероятно...
～かもしれません - может быть...
```

#### 報連相 (Houkoku・Renraku・Soudan)
```
報告 (houkoku) - отчеты:

定期報告:
週報 (shuuhou) - еженедельный отчет
月報 (geppo) - ежемесячный отчет
四半期報告 (shihanki houkoku) - квартальный отчет

報告の構造:
1. 概要 (gaiyou) - краткое изложение
2. 詳細 (shoushitsu) - детали
3. 問題点 (mondai-ten) - проблемы
4. 今後の予定 (kongo no yotei) - планы на будущее

例文:
売上高は前月比5%増となりました。
- Объем продаж увеличился на 5% по сравнению с прошлым месяцем.

新規顧客獲得数は目標を上回りました。
- Количество новых клиентов превысило цель.

連絡 (renraku) - информирование:

緊急連絡:
至急 (shikyuu) - срочно
重要 (juuyou) - важно
確認要 (kakunin you) - требует подтверждения

連絡手段:
メール (e-mail) - электронная почта
電話 (denwa) - телефон
社内チャット (shanai chat) - внутренний чат

相談 (soudan) - консультации:

相談のタイミング:
- 問題が発生した時点で - когда возникает проблема
- 判断に迷った時 - когда сомневаетесь в решении
- 新しいアイデアがある時 - когда есть новые идеи

相談の仕方:
お忙しい中恐れ入りますが、ご相談があります。
- Извините за беспокойство, но у меня есть вопрос для консультации.

どのように進めたらよいでしょうか。
- Как лучше всего продолжить?
```

### 契約書・提案書作成

#### 契約書の基本用語
```
契約の種類:

売買契約 (baibai keiyaku) - договор купли-продажи
業務委託契約 (gyoumu itaku keiyaku) - договор подряда
秘密保持契約 (himitsu hoji keiyaku) - соглашение о неразглашении
ライセンス契約 (license keiyaku) - лицензионное соглашение

基本条項:

第1条 (目的) - статья 1 (цель)
本契約は、甲乙間における～を目的とする。

第2条 (期間) - статья 2 (период)
本契約の有効期間は～から～までとする。

第3条 (対価) - статья 3 (вознаграждение)
甲は乙に対し、～を支払うものとする。

第4条 (責任) - статья 4 (ответственность)
甲及び乙は、～について責任を負う。

第5条 (解除) - статья 5 (расторжение)
本契約は、以下の場合に解除できる。

法的表現:

みなす - считать, рассматривать как
に限る - ограничиваться
を除く - за исключением
に関わらず - независимо от
を条件として - при условии

例文:
甲の承諾を得た場合に限り、本契約を変更することができる。
- Изменение данного договора возможно только при согласии стороны А.
```

6. リスクと対策:
想定されるリスク

代替案

### ビジネスシミュレーション

#### ケーススタディ 1: 新製品発表会
```
シナリオ:
あなたの会社が新製品を開発しました。
記者発表会で製品を紹介することになりました。

準備項目:
1. プレゼンテーション資料作成
2. 質疑応答の準備
3. メディア対応

実践内容:

開会挨拶:
本日はお忙しい中、弊社の新製品発表会にお越しいただき、
ありがとうございます。

製品紹介:
従来品と比較して、性能が50%向上しています。
環境に配慮した設計となっております。
価格は従来品と同程度に設定いたします。

質疑応答例:

Q: 競合他社との差別化ポイントは？
A: 独自の技術により、省エネ性能を大幅に向上させました。

Q: 販売目標は？
A: 初年度10万台の販売を目指しています。

Q: 海外展開の予定は？
A: 来年からアジア市場への展開を計画しています。

フォローアップ:
発表後のメディア対応
プレスリリースの配布
販売店への説明会実施
```

#### ケーススタディ 2: 国際会議での交渉
```
シナリオ:
海外企業との合弁事業について交渉します。
文化的違いを考慮した交渉が必要です。

交渉ポイント:
1. 出資比率
2. 経営権
3. 技術移転
4. 利益配分

交渉プロセス:

初回会議:
まず、お互いの会社について理解を深めましょう。
長期的なパートナーシップを構築したいと考えています。
双方にメリットのある提案をしたいと思います。

条件提示:
弊社としては、以下の条件を提案いたします。
技術面での協力を重視したいと考えています。
市場開拓については、共同で取り組みたいと思います。

妥協案:
そちらのご意見も十分理解できます。
中間的な解決策を考えてみませんか。
段階的な実施はいかがでしょうか。

合意形成:
基本的な方向性について合意できました。
詳細は法務部門同士で検討していただけませんか。
来月までに正式契約を締結しましょう。

文化的配慮:
相手の意見を十分に聞く
急かさずに時間をかける
win-winの関係を強調
長期的視点の重要性
```
